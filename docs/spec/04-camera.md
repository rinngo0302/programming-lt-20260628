# カメラ仕様

v1ではプレイヤーのカートを追従する三人称カメラに加えて、コース全体とレーサーの位置を表示するミニマップを実装する。

## 挙動方針

- カメラはプレイヤーのカートの**後方・上方**に位置し、カートの進行方向を追従する
- カートの回転に対してカメラの回転は適度に遅延させ(完全追従ではなく緩やかに補間)、急なステアリングでも視点がカクつかないようにする
- カート同士の衝突やアイテムによるスピン中もカメラは大きく揺れすぎないようにする(カメラの回転はカートの実際の回転と完全同期させない)

## パラメータ(初期値の目安、実装時に調整)

| パラメータ | 初期値の目安 |
|---|---|
| カートからの後方距離 | 5〜6m |
| カートからの高さ | 2〜3m |
| 注視点 | カートのやや前方(進行方向) |
| 追従の補間速度 | 位置・回転ともに緩やかな補間(Lerp/Slerp) |

具体的な数値はプレイテストで調整する前提とする。

## CPUカメラ

CPUのカートにはカメラを割り当てない(画面に表示する必要がないため)。

## ミニマップ

- コース全体を上空から見下ろす直交投影カメラ(`MinimapCamera`)を、画面の隅(右上)に小さなビューポートで重ねて表示する
- レーサー(プレイヤー・CPU)を色分けして見分けやすくする(プレイヤー: 緑、CPU: 青、アイテムボックス: 赤)
- チェックポイントなど見た目を持たないオブジェクトはミニマップ上にも表示されない(描画コンポーネントが無いため)
