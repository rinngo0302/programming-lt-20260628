# ゲーム概要

## ジャンル

マリオカート風のレーシングゲーム。プレイヤー1人 + CPU3体の合計4台でコースを周回し、順位を競う。

## v1のスコープ

- コース: 1つ
- レーサー: プレイヤー1台 + CPU3台(合計4台)
- ラップ数: 3ラップ
- 操作: キーボードのみ
- カメラ: 三人称追従カメラのみ
- アイテム: キノコ・みどり甲羅・バナナの3種類([05-items.md](05-items.md)参照)
- ドリフトなどの高度な操作要素は含めない(加速・ブレーキ・ステアリングのみ)

上記以外の要素(複数コース、キャラ/カート選択、対戦相手の増減、ドリフトなど)はv1では実装しない。将来的な拡張候補として扱う。

## シーン構成と遷移

```
タイトルシーン → ゲームシーン → リザルトシーン → ゲームシーン → リザルトシーン → ...
```

- タイトルシーンからスタートするとゲームシーンに遷移する
- ゲームシーンでレース(3ラップ)が終了するとリザルトシーンに遷移する
- リザルトシーンから「もう一度プレイ」を選ぶとゲームシーンに戻る(タイトルへ戻る選択肢も用意する。詳細は[01-scenes.md](01-scenes.md))

詳細はそれぞれ以下を参照:

## アーキテクチャ前提

実装時は `CLAUDE.md` の開発規約に従う。特に以下を前提として各仕様を記述する。

- UIはMVP(Model-View-Presenter)パターン
- イベント・値の監視はR3(`UnityEvent`は使わない)
- コース定義・アイテム設定などのゲームデータは`ScriptableObject`で持つ
- namespaceは省略可
