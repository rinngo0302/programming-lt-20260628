# アイテム仕様

v1で実装するアイテムは3種類: **キノコ** / **みどり甲羅** / **バナナ**。

## アイテムボックス

- コース上の固定位置に複数個配置する(同時に複数台が取得できるよう、コース幅に応じて2〜3個並べて設置することを想定)
- レーサーがアイテムボックスに接触すると、3種類からランダムで1つ抽出してそのレーサーに付与する
- アイテムボックスは取得されると一時的に消え、一定時間後(目安: 5秒)に再出現する
- 各レーサーが同時に保持できるアイテムは**1つまで**。既にアイテムを保持している場合は新たに取得できない(ボックスに当たっても何も起きない、または取得自体をブロックする)

## アイテム効果

### キノコ(加速)

| 項目 | 値の目安 |
|---|---|
| 効果 | 使用した瞬間から一定時間、速度が上昇する |
| 持続時間 | 2秒程度 |
| 速度上昇率 | 通常速度の1.5倍程度 |
| 対象 | 使用者本人のみ |

### みどり甲羅(前方攻撃)

| 項目 | 値の目安 |
|---|---|
| 効果 | 使用者の前方にまっすぐ発射される弾。コース上の壁で反射せず、命中した相手をスピンさせる(後述のスタン) |
| 飛行速度 | カートの最高速度より速い一定速度 |
| 有効距離/時間 | 命中または一定時間(目安: 3秒)経過で消滅 |
| 命中判定 | 発射者の前方にいる他のレーサーとの当たり判定 |

### バナナ(設置トラップ)

| 項目 | 値の目安 |
|---|---|
| 効果 | 使用者の現在位置(やや後方)にその場で設置される。誰か(設置者含むかは要検討、v1では設置者以外と判定)が踏むとスピン(スタン)する |
| 持続時間 | 誰かが踏むまで、またはコース上に長時間残り続けないよう一定時間(目安: 20秒)で自然消滅する |

## スピン(スタン)効果の共通仕様

アイテム(みどり甲羅・バナナ)を受けたレーサーは以下の状態になる。

- 一定時間(目安: 1.5秒)操作不能になる
- 速度が大きく低下する(0付近まで減速)
- スタン演出として回転アニメーション等を再生する(見た目は実装時に検討)

## アイテム使用操作

- プレイヤー: [03-controls.md](03-controls.md) の `Space` キーで使用
- CPU: [06-cpu-ai.md](06-cpu-ai.md) の方針に従いランダムなタイミングで使用

## アーキテクチャメモ

- アイテムの種類と効果パラメータ(持続時間・速度倍率など)は `ItemData` のような `ScriptableObject` で定義し、コード変更なしに調整できるようにする
- 「保持中のアイテム」「スタン状態かどうか」はレーサーごとの Model(`KartModel`等)が `ReactiveProperty` として保持する
